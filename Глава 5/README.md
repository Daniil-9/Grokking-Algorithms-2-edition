# Глава 5. Хеш-таблицы

## Хеш-функции  

 - **Хеш-функция** представляет собой функцию, которая получает строку и возвращает число.

**Требования** к ней:  
 - Она должна быть **последовательной**, то есть возвращала один и тот же результат для одинаковых входных данных.  
 - Разным словам должны соответствовать разные числа.

## Коллизии

 - **Коллизия**: двум ключам назначается один элемент массива.
 - Это плохо.

## Быстродействие

|          | Средний случай | Худший случай |
|----------|----------------|---------------|
| Поиск    | O(1)           | O(n)          |
| Вставка  | O(1)           | O(n)          |
| Удаление | O(1)           | O(n)          |

## Коэффициент заполнения 

 - Коэф. заполнения = кол-во элементов в хеш-таблице / общее кол-во элементов
