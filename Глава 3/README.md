# Глава 3. Рекурсия

## Рекурсия   

 - это вызов функцией самой себя

## Базовый случай и рекурсивный случай

Каждая рекурсивная функия состоит из двух частей: **базового** и **рекурсивного** случая  
 - в рекурсивном случае функция вызывает сама себя  
 - в базовом случае функция себя не вызывает, чтобы предотвратить зацикливание

[Пример](Countdown.py)

## Стек  

 - это простая структура данных  
 - работает по принципу LIFO (Last in, First Out / последним пришел, первым и ушел)

## Стек вызовов  

 - когда вы вызываете функцию из другой функции, вызывающая функция приостанавливается в частично завершенном состоянии
