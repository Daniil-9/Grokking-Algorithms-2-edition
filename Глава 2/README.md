# Глава 2. Сортировка выбором

## Как работает память

 - Она представляет собой нечто вроде шкафа со множеством ящиков, и у каждого ящика есть адрес

## Массивы и связные списки

 - **Массив** хранит данные непрерывно. Чтение мгновенно, а запись требует сдвига  
 - **Связный список** хранит данные в узлах со ссылками. Запись проста, а чтение требует обхода

## Какая структура данных используется чаще: массивы или списки

 - Массивы используются чаще, чем связанные списки, за исключением особоых случаев

## Сортировка выбором

 - Вы хотите отсортировать список по убыванию. Надо пройтись по списку и найти самый большой элемент. Этот элемент добавляем в новый список. Потом то же самое для следующих  
 - Сложность: O(n^2)
