# Глава 6. Поиск в ширину

## Что такое граф?  

 - **Граф** моделирует набор связей между объектами  
 - Каждый граф состоит из узлов и ребер  
 - Узел может быть соединен с другими узлами. Это называется внутренние (от кого) и внешние (к кому) соседи

## Поиск в ширину

Помогает ответить на вопросы двух типов:  
 - существует ли путь от узла A к узлу B?  
 - как выглядит кратчайший путь от узла A к узлу B?

## Очередь  

 - **Очередь** относится к категории структур данных FIFO: First In, First Out ("Первым вошел, первым вышел")
   
Только две операции:  
 - постановка в очередь  
 - извлечение из очереди
